# IntelliJ Method Numbers Plugin

![Build](https://github.com/michaelomichael/intellij-method-numbers-plugin/workflows/Build/badge.svg)
[![Version](https://img.shields.io/jetbrains/plugin/v/26566.svg)](https://plugins.jetbrains.com/plugin/26566)
[![Downloads](https://img.shields.io/jetbrains/plugin/d/26566.svg)](https://plugins.jetbrains.com/plugin/26566)

<!-- Plugin description -->
Displays sequential numeric hints beside every method in the file.

This lets you can more easily keep track of large test classes.

It currently supports basic Java and Kotlin.

![preview](https://raw.githubusercontent.com/michaelomichael/intellij-method-numbers-plugin/refs/heads/main/.github/readme/example-java-dark.png)

<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "intellij-method-numbers-plugin"</kbd> >
  <kbd>Install</kbd>
  
- Using JetBrains Marketplace:

  Go to [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/MARKETPLACE_ID) and install it by clicking the <kbd>Install to ...</kbd> button in case your IDE is running.

  You can also download the [latest release](https://plugins.jetbrains.com/plugin/MARKETPLACE_ID/versions) from JetBrains Marketplace and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>

- Manually:

  Download the [latest release](https://github.com/michaelomichael/intellij-method-numbers-plugin/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
